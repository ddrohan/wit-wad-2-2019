Web App Development

David Drohan BSc., MSc, WIT

    Node JS

    Node - Part 1

    Here we discuss Node in general, the frameworks available to the app developer and the different approaches that can be followed in the design and development of a Node web app.

    Node - Part 2

    Here we investigate some of the more advanced features used in the design of Node web apps and focus on a particular Framework - Express.

    Lab-01

    In this lab we're going to build a very simple Node Web App using the WebStorm IDE. If you haven't done so already, get it set up by following the WebStorm IDE Setup. This web app won't have much functionality but it will take you through the process of building a Node Web app using WebStorm.

    Lab-02

    In this lab we will be using our WebStorm IDE to build a simple Node/Express Server to initially act as a Web Service APi which we will test manually, and eventually it will be our back-end to a full MEAN Stack Web App.

    Node, MongoDB & Mongoose

    Node - Part 3

    Here we explore the use of NOSQL databases when used in Node web apps. We focus on MongoDB and Mongoose to build persistence into our Case Study web app.

    Lab-03

    In this lab we will be introducing some proper persistence, through a MongoDB backend, initially running locally, and then eventually deploying to www.mlab.com so that we can have remote mongo support.

    Deploy-to-heroku

    As a related exercise (and a useful one!) you should complete this lab in the context of your own Web App, and deploy your Node Server to Heroku.

    Vue JS - Routing, Directives & Event Handling

    Routing

    Here we discuss the different approaches available to the developer for implementing url routing in a Vue web app, from very basic in-line routing to advanced component based systems.

    Directives

    Directives are a fundamental part of any Vue web app. In this section we look at the different types of directives that the developer can use to enhance the functionality and usefulness of a Vue app.

    Event Handling

    Event Handling is pretty much a given for any useful web app. Here we discuss Vue Event Handling in general, and look at specific examples implemented in our Case Study.
